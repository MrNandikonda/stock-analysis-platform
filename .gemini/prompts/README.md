# Gemini Prompt Library

These prompts are reusable starting points for future Gemini Code Assist sessions in this repo. They are not product configuration files; copy or reference the relevant prompt when starting a task.

Use every prompt with the operating rules in `GEMINI.md`.

## Available Prompts

- `MASTER_ORCHESTRATOR_PROMPT.md` - broad repo tasks and cross-stack coordination.
- `FEATURE_CHANGE_PROMPT.md` - adding or changing application behavior.
- `BUGFIX_PROMPT.md` - investigating and fixing a defect.
- `MARKET_DATA_PROMPT.md` - provider, quote/history/options/fundamentals, freshness, and fallback work.
- `QUANT_RISK_PROMPT.md` - screener, indicators, portfolio risk, and AI financial-analysis logic.
- `DOCKER_WINDOWS_PROMPT.md` - Docker Compose, nginx, env, Cloudflare Tunnel, and Windows laptop deployment.
- `QA_REVIEW_PROMPT.md` - final review, validation, security, and regression checks.
